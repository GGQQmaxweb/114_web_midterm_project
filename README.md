## 📝 專案簡介

本專案是一個以 **HTML Canvas** 製作的貪吃蛇小遊戲，提供多種難度、配色主題，以及本地化的 **分數排行榜**。
遊戲包含方向鍵與 WASD 操作提示，結束後可輸入姓名與 Email，並將分數記錄在 LocalStorage。

## ⭐ 功能特色

### 🎮 遊戲功能

1. **多種難度選擇**（簡單、中等、困難、地獄、無限速）
2. **多種配色主題**（蛇與果實可切換顏色）
3. **方向鍵 / WASD 控制**
4. **食物不會生成在蛇身上**
5. **即時分數顯示**

### 🧪 表單與資料功能

1. **分數提交表單**含姓名與 email  (自動驗證)
2. **自動填入上次輸入的姓名/Email**
3. **依難度分類的分數排行榜（LocalStorage）**
4. **最佳成績排序排名**

### 🖥 UI / UX

1. **開始畫面（難度+主題設定）**
2. **遊戲開始前須按方向鍵 → 才會開始**
3. **操作提示圖片 Overlay**
4. **RWD 自適應設計**

## 🚀 如何開始

1. 開啟網頁
   → 選擇難度與配色
   → 按下「開始遊戲」

2. 在 Canvas 出現後
   → 按任意方向鍵 / WASD 開始

3. 遊戲結束後
   → 填寫表單並送出分數
   → 觀看排行榜
